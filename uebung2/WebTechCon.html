Informationen zu WebTech Con

So bietet die Konferenz mit ihren über 90 Vorträgen für Webdesigner, -entwickler, Software-Architekten und Projektleiter ein unvergleichliches Erlebnis. Gemeinsam mit der parallel stattfindenden International PHP Conference bietet sie eine Informationsdichte, die ihres gleichen sucht.
Abonnieren Sie den Newsletter und folgen Sie uns auf Twitter (@WebTechCon), um über alle Neuigkeiten informiert zu werden..





Die WebTech Conference hat das gesamte Spektrum an Technologien zum Thema, das bedeutend für den Geschäftserfolg im Web ist. Dazu gehören HTML5 und JavaScript ebenso wie Architekturthemen, Cloud Computing oder Web Security. Einen weiteren Schwerpunkt bilden Mobile Websites und Anwendungen für Smartphones und Tablets.
